Alignment and distribution tools help you precisely arrange multiple elements on the canvas. Select two or more elements to access these options in the right sidebar.

## Horizontal Alignment

Align selected elements along the horizontal axis:

| Icon | Tool | Description |
|------|------|-------------|
| ![Align Left](https://raw.githubusercontent.com/Diskyeth/FigClank-Studio/main/Docs%26Guides/doc_resources/icons/align-left.svg) | **Align Left** | Align all selected elements to the left edge of the leftmost element |
| ![Align Center](https://raw.githubusercontent.com/Diskyeth/FigClank-Studio/main/Docs%26Guides/doc_resources/icons/align-center.svg) | **Align Center (Horizontal)** | Align all selected elements to the horizontal center of the selection |
| ![Align Right](https://raw.githubusercontent.com/Diskyeth/FigClank-Studio/main/Docs%26Guides/doc_resources/icons/align-right.svg) | **Align Right** | Align all selected elements to the right edge of the rightmost element |

Align selected elements along the vertical axis:

| Icon | Tool | Description |

| ![Align Middle](https://raw.githubusercontent.com/Diskyeth/FigClank-Studio/main/Docs%26Guides/doc_resources/icons/align-middle.svg) | **Align Middle (Vertical)** | Align all selected elements to the vertical center of the selection |

## Distribution

Distribute three or more elements with equal spacing between them. The first and last elements remain fixed, while the elements in between are repositioned to create even gaps.

| Icon | Tool | Description |
|------|------|-------------|
| ![Distribute Horizontally](https://raw.githubusercontent.com/Diskyeth/FigClank-Studio/main/Docs%26Guides/doc_resources/icons/distribute-horizontally.svg) | **Distribute Horizontally** | Space elements evenly along the horizontal axis with equal gaps between bounding boxes |
| ![Distribute Vertically](https://raw.githubusercontent.com/Diskyeth/FigClank-Studio/main/Docs%26Guides/doc_resources/icons/distribute-vertically.svg) | **Distribute Vertically** | Space elements evenly along the vertical axis with equal gaps between bounding boxes |

## Usage Tips

- **Alignment:** Select 2 or more elements to enable alignment buttons
- **Distribution:** Select 3 or more elements to enable distribution buttons
- **Undo:** All alignment and distribution actions can be undone with `Cmd/Ctrl + Z`
- **Mixed sizes:** Distribution creates equal gaps regardless of element sizes
