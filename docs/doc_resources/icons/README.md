# FAQ Icons for GitBook

These SVGs match the icons used in the FigClank app FAQ pages. Use them in GitBook so docs match the in-app help.

## How to use in GitBook

- **Image block:** Upload the SVG to your GitBook space and insert the image, or reference the file path if GitBook is synced from this repo.
- **Markdown:** `![Alt text](icons/align-left.svg)` (adjust path to your GitBook asset path).
- **HTML:** `<img src="icons/align-left.svg" alt="Align Left" width="24" height="24" />`

If you host the repo (e.g. GitHub Pages), use the full URL to the raw file for `src`.
