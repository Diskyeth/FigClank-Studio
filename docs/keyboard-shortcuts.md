## Tool Selection

Select tools using these single-letter shortcuts (no modifier keys required):

| Key | Tool | Description |
|-----|------|-------------|
| `V` | Cursor/Select | Select and move objects |
| `F` | Frame | Create frame containers |
| `T` | Text | Add text elements |
| `P` | Vector | Draw vector paths |
| `O` | Circle | Create circle shapes |
| `R` | Square | Create square/rectangle shapes |
| `K` | Resize | Open resize/scale modal for the selected node (single selection only) |

## Viewport & Zoom

Control the canvas viewport and zoom level:

| Key | Action | Description |
|-----|--------|-------------|
| `+` or `=` | Zoom In | Zoom in at mouse position (or canvas center) |
| `-` or `_` | Zoom Out | Zoom out at mouse position (or canvas center) |
| `0` | Fit to Screen | Fit the entire design to the viewport |
| `1` | Reset Zoom | Reset zoom to 100% and center at origin |
| `Z` + Drag | Zoom by Dragging | Hold Z key and drag to zoom in/out |

## Rulers

Control ruler visibility and create guide lines:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Shift + R` | Toggle Rulers | Show or hide rulers and guide lines |
| Drag from top edge | Create Horizontal Guide | Drag from the top ruler to create a horizontal guide line |
| Drag from left edge | Create Vertical Guide | Drag from the left ruler to create a vertical guide line |

## Edit Actions

Undo, redo, copy, paste, and delete operations:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd + Z` (Mac) / `Ctrl + Z` (Win) | Undo | Undo the last action |
| `Cmd + Shift + Z` (Mac) / `Ctrl + Shift + Z` (Win) | Redo | Redo the last undone action |
| `Cmd + C` (Mac) / `Ctrl + C` (Win) | Copy | Copy selected objects, text, or frames |
| `Cmd + V` (Mac) / `Ctrl + V` (Win) | Paste | Paste copied objects at a new position (offset by 10px) |
| `Cmd + G` (Mac) / `Ctrl + G` (Win) | Group | Group selected nodes into a frame (at least 2 nodes) |
| `Option + Cmd + K` (Mac) / `Alt + Ctrl + K` (Win) | Create Component | Convert selected nodes to components |
| `Delete` or `Backspace` | Delete | Delete selected nodes |

**Note:** Shortcuts are disabled when typing in input fields or text areas, or when editing text nodes on the canvas.

## Mouse Interactions

While not keyboard shortcuts, these mouse interactions are available:

- **Click:** Select objects
- **Drag:** Move selected objects or pan the canvas (when no selection)
- **Double-click:** Edit text nodes
- **Scroll wheel:** Zoom in/out at mouse position
- **Right-click:** Context menu (shows available shortcuts)
